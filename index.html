🔥 Configurar Firebase (armazenamento em nuvem)

Siga os passos abaixo uma única vez (~15 min) para ter dados em tempo real em qualquer dispositivo.

Passo 1 — Firebase
1. Acesse firebase.google.com → Criar projeto (ex: sgi-loja06)
2. No projeto, clique em "Firestore Database" → Criar banco → Modo de teste
3. Vá em Configurações ⚙️ → Configurações do projeto → Adicionar app Web
4. Copie o bloco firebaseConfig = { apiKey: ... }

Passo 2 — GitHub Pages (URL permanente)
1. Crie conta em github.com → Novo repositório público (ex: sgi-loja06)
2. Cole o config Firebase no início do arquivo HTML (onde está COLE_AQUI)
3. Faça upload do arquivo HTML pro repositório
4. Vá em Settings → Pages → Branch: main → Save
5. Acesse: https://seuusuario.github.io/sgi-loja06/

Passo 3 — Regras do Firestore
No Firebase, vá em Firestore → Regras → cole:
rules_version = '2';
service cloud.firestore {
  match /databases/{db}/documents {
    match /{doc=**} { allow read, write: if true; }
  }
}
